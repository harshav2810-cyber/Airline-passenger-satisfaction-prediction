# ✈️ Airline Passenger Satisfaction Prediction

## 📌 Project Overview

This project is a **Machine Learning web application** that predicts whether an airline passenger is likely to be **Satisfied** or **Neutral/Dissatisfied** based on passenger information, travel details, and service ratings.

The application is built using **Python**, **Scikit-learn**, and **Streamlit**, providing an interactive interface for real-time predictions.

---

## 🚀 Features

* Predicts passenger satisfaction using a trained Machine Learning model.
* User-friendly web interface built with Streamlit.
* Supports passenger details such as age, gender, travel type, and customer type.
* Includes service quality ratings and travel information as inputs.
* Displays instant prediction results.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Joblib
* Streamlit

---

## 🤖 Machine Learning Model

* **Algorithm:** Random Forest Classifier
* **Task:** Binary Classification
* **Target Variable:** Passenger Satisfaction

  * Satisfied
  * Neutral or Dissatisfied

## ▶️ How to Run

1. Clone the repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Start the Streamlit application:

```bash
streamlit run app.py
```

---

## 📷 Application

The Streamlit application allows users to enter passenger details and instantly receive a prediction indicating whether the passenger is likely to be satisfied with the airline experience.
